---
title: "Quickstart"
description: "Crie um token e faça sua primeira chamada à API em menos de 5 minutos."
---

<Steps>
  <Step title="Verifique seu plano">
    O recurso de tokens de API só está disponível em planos pagos (Lite, Pro, Scale). Acesse **Configurações → Planos** para ver o seu.

    <Warning>
      Se seu plano não inclui API, o item de menu aparecerá com cadeado e qualquer requisição retornará erro `402 Payment Required`.
    </Warning>
  </Step>

  <Step title="Crie um token de API">
    1. Logue como **Super Admin** no painel da sua tenant.
    2. Acesse **Configurações → Tokens de API**.
    3. Clique em **Novo token**.
    4. Preencha:
       - **Nome** — identifique a integração (ex: `n8n produção`).
       - **Descrição** (opcional).
       - **Escopos** — marque apenas o necessário (princípio do menor privilégio).
       - **Expira em** (opcional) — recomendado para integrações temporárias.
       - **IPs permitidos** (opcional) — restrinja a origem.
    5. Clique em **Criar token**. O token aparece **uma única vez** — copie imediatamente.

    <Note>
      Tokens começam com `aut_` para serem detectados por scanners de secrets do GitHub/GitLab caso vazem.
    </Note>
  </Step>

  <Step title="Faça sua primeira chamada">
    Substitua `aut_xxx...` pelo token copiado e `seu-dominio` pelo domínio da tenant.

    <CodeGroup>
    ```bash cURL
    curl https://seu-dominio/api/v1/me \
      -H "Authorization: Bearer aut_xxx..."
    ```

    ```javascript Node.js
    const response = await fetch('https://seu-dominio/api/v1/me', {
      headers: {
        'Authorization': 'Bearer aut_xxx...',
      },
    });
    const data = await response.json();
    console.log(data);
    ```

    ```python Python
    import requests

    response = requests.get(
        'https://seu-dominio/api/v1/me',
        headers={'Authorization': 'Bearer aut_xxx...'},
    )
    print(response.json())
    ```

    ```php PHP
    $response = Http::withToken('aut_xxx...')
        ->get('https://seu-dominio/api/v1/me');

    print_r($response->json());
    ```
    </CodeGroup>
  </Step>
