As **tarefas** permitem pedir entregas dos alunos — texto, arquivo, link ou vídeo — e acompanhar quem fez. São a base do acompanhamento ativo em turmas.

Gerencie em **Admin → Turmas → [Turma] → Tarefas**.

```
/admin/classes/{turma}/assignments
```

## Criando uma tarefa

| Campo | Descrição |
|---|---|
| **Título** | Nome da tarefa. Obrigatório. |
| **Descrição** | Resumo da tarefa. |
| **Instruções** | Orientações detalhadas (aceita formatação). |
| **Tipo de entrega** | Veja os tipos abaixo. |
| **Módulo** / **Lição** | Vincula a tarefa a um ponto do conteúdo (opcional). |
| **Prazo** | Data e hora limite (opcional). |
| **Permitir reentrega** | Aluno pode reenviar após o feedback. |
| **Obrigatória** | Conta para o acompanhamento de conclusão. |
| **Nota máxima** | Pontuação máxima (1 a 100), se for avaliar com nota. |

## Tipos de entrega

| Tipo | O que o aluno envia |
|---|---|
| **Texto** | Resposta escrita. |
| **Arquivo** | Upload de arquivo. |
| **Link** | URL. |
| **URL de vídeo** | Link de vídeo. |
| **Qualquer** | Qualquer um dos formatos acima. |

## Reentrega

Com **Permitir reentrega** ativado, o aluno pode reenviar após receber feedback que peça revisão — útil para tarefas que exigem ajustes até a aprovação.

Depois que os alunos enviam, avalie na [fila de revisão](/help-center/pt/turmas/revisando-entregas).
