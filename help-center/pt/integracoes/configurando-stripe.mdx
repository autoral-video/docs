O Stripe é o principal gateway de pagamento da plataforma. Ele permite processar vendas em diversas moedas e métodos de pagamento internacionais, sendo ideal para alcançar alunos de qualquer país com uma experiência de checkout otimizada.

A plataforma utiliza o **Stripe Connect** para vincular sua conta Stripe via OAuth. Isso significa que você conecta com um único clique — sem chaves de API para copiar, sem webhooks para configurar manualmente. Tudo é configurado automaticamente.

## O que você vai precisar

Antes de começar, certifique-se de ter:

- Uma conta no Stripe — crie em [stripe.com](https://stripe.com)
- Acesso ao **Painel do Stripe** (dashboard.stripe.com)

## Passo 1: Acesse as configurações do Stripe

1. Acesse **Admin → Configurações → Stripe**

Você verá um painel de conexão com o botão **Conectar com Stripe**.

## Passo 2: Conecte sua conta Stripe

1. Clique em **Conectar com Stripe**
2. Você será redirecionado para a página de autorização do Stripe
3. Faça login na sua conta Stripe (ou crie uma)
4. Revise as permissões solicitadas e clique em **Authorize access**
5. Você será redirecionado de volta à plataforma

Após o redirecionamento, o painel exibirá um badge de confirmação **Stripe conectado** junto ao identificador da sua conta.

<Note>Suas chaves de API e a configuração do webhook são definidas automaticamente — nenhuma credencial precisa ser inserida manualmente.</Note>

## Passo 3: Ative o Stripe

Após a conexão, ative o toggle **Ativar Stripe** para habilitá-lo como forma de pagamento nos seus checkouts.

<Note>O Stripe só estará disponível no checkout após estar conectado e ativado.</Note>

## Gerenciando a conexão

### Desativar o Stripe

Você pode desativar o Stripe a qualquer momento pelo toggle **Ativar Stripe**. Sua conta permanece conectada — é possível reativar sem precisar passar pelo fluxo OAuth novamente.

### Desconectar a conta

Para remover completamente a integração com o Stripe:

1. Acesse **Admin → Configurações → Stripe**
2. Clique em **Desconectar**
3. Confirme a ação

Isso revoga a autorização OAuth e remove todas as credenciais armazenadas. Para usar o Stripe novamente, será necessário passar pelo fluxo de conexão uma nova vez.

## Resumo das ações

| Ação | Onde |
|---|---|
| Conectar conta Stripe | Admin → Configurações → Stripe → **Conectar com Stripe** |
| Ativar / desativar no checkout | Admin → Configurações → Stripe → toggle **Ativar Stripe** |
| Desconectar conta | Admin → Configurações → Stripe → **Desconectar** |
