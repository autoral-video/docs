title: "Members and moderators"
description: "Add and remove members from invite-only groups, promote moderators, and understand the community's roles and permissions."
---

In groups in **Invite only** mode, you manually control who participates. In any group, you can assign **moderators**.

Go to **Admin → Community → Groups → [Group] → Members**.

```
/admin/community/groups/{group}/members
```

<Note>
The **Members** tab only appears for groups in **Invite only** mode. In **open** or **linked** groups, access is automatic and there is no manual member list.
</Note>

## Adding members

1. Search for the user by **name or email**.
2. Select them from the list.
3. Click **Add**.

## Promoting moderators

For each member, use **Make moderator** / **Remove moderation**. A **Moderator** badge identifies whoever has the role.

## Removing members

Use the remove button (×) next to the member. They lose access to the group.

## Roles and permissions

| Action | Member | Moderator | Administrator |
|---|:---:|:---:|:---:|
| Post, comment, react, and vote in polls | ✓ | ✓ | ✓ |
| Hide/delete **their own** posts and comments | ✓ | ✓ | ✓ |
| Pin posts (up to 3 per group) | — | ✓ | ✓ |
| Hide/delete posts and comments **from anyone** | — | ✓ | ✓ |

<Note>
**Class mentors** linked to the group automatically become moderators — they don't need to be added manually.
</Note>
