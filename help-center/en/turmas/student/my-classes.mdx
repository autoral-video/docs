title: "My classes (student)"
description: "Access your classes, view the overview, upcoming events and pending assignments in the member area."
---

As a student, all the classes you're enrolled in are gathered in **My classes**.

```
/member/my-classes
```

Each class appears as a card with the name, the course and the upcoming events.

## Class overview

When you open a class, the overview shows:

- Class information
- **Upcoming events** (live sessions, deadlines, meetings)
- **Pending assignments** with an approaching deadline

From there you navigate the class sections:

| Section | What it is |
|---|---|
| **Roadmap** | Content map, showing what's been released. See [Content roadmap](/help-center/en/turmas/student/content-roadmap). |
| **Calendar** | Class events. See [Calendar](/help-center/en/turmas/student/calendar). |
| **Assignments** | Submissions to complete. See [Submitting assignments](/help-center/en/turmas/student/submitting-assignments). |
| **Certificate** | Your progress toward the certificate. See [My certificate](/help-center/en/turmas/student/my-certificate). |
