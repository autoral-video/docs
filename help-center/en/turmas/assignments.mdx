**Assignments** let you request submissions from students — text, file, link or video — and track who has completed them. They are the foundation of active support in classes.

Manage them in **Admin → Classes → [Class] → Assignments**.

```
/admin/classes/{class}/assignments
```

## Creating an assignment

| Field | Description |
|---|---|
| **Title** | Name of the assignment. Required. |
| **Description** | Summary of the assignment. |
| **Instructions** | Detailed guidance (supports formatting). |
| **Submission type** | See the types below. |
| **Module** / **Lesson** | Links the assignment to a point in the content (optional). |
| **Deadline** | Cutoff date and time (optional). |
| **Allow resubmission** | Student can resubmit after feedback. |
| **Required** | Counts toward completion tracking. |
| **Maximum score** | Maximum score (1 to 100), if you grade with a score. |

## Submission types

| Type | What the student submits |
|---|---|
| **Text** | Written answer. |
| **File** | File upload. |
| **Link** | URL. |
| **Video URL** | Video link. |
| **Any** | Any of the formats above. |

## Resubmission

With **Allow resubmission** enabled, the student can resubmit after receiving feedback that requests changes — useful for assignments that need adjustments until approval.

## Next step

Once students submit, grade them in the [review queue](/help-center/en/turmas/reviewing-submissions).
