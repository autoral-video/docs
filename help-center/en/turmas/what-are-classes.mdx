---
title: "What are classes"
description: "Understand the concept of a class (cohort): sell with a start date, limited seats, a dedicated community, and active student tracking."
---

A **class** turns a course into an experience with a **date, seats, and a group**. Instead of each student joining alone and studying in isolation, everyone starts together, follows a common schedule, and takes part in a dedicated community — the **cohort** format.

It is the recommended approach for mentors and educators who actively support their students.

## Class vs. open access

| | Open access | Class (cohort) |
|---|---|---|
| Start | immediate, individual | common date |
| Seats | unlimited | can be limited |
| Content | all unlocked | released on a schedule (drip) |
| Community | general | class group |
| Tracking | passive | engagement, assignments, certificates |

## The lifecycle of a class

A class goes through phases, controlled by the **Status** field:

| Status | Meaning |
|---|---|
| **Draft** | In preparation, hidden from the public. |
| **Open** | Accepting enrollments. |
| **Enrollment open** | Active enrollment window. |
| **In progress** | Class running with students. |
| **Closed** | No longer accepting new enrollments. |
| **Cancelled** | Class cancelled; enrollments blocked. |

## Dates that control the class

| Date | Purpose |
|---|---|
| **Start** | The zero point for content release (drip). |
| **End** | Expected end of the class (informational). |
| **Opens on / Closes on** | Enrollment window. With the **strict window**, checkout is blocked outside this period. |
