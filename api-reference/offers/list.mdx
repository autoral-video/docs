---
title: "Listar ofertas"
api: "GET /v1/offers"
description: "Lista paginada (length-aware) de ofertas (preços/planos)."
---

**Escopo:** `offers:read`

Ofertas representam o **preço e formato de cobrança** de um produto. Um produto pode ter múltiplas ofertas (vitalício, mensal, anual, etc).

## Query params

<ParamField query="per_page" type="integer" default="25">1-100.</ParamField>
<ParamField query="page" type="integer" default="1">Página.</ParamField>
<ParamField query="product_id" type="integer">Filtra por produto.</ParamField>
<ParamField query="active" type="boolean">Filtra ativos/inativos.</ParamField>

## Response

<ResponseField name="data[].id" type="integer">ID.</ResponseField>
<ResponseField name="data[].product_id" type="integer">FK do produto.</ResponseField>
<ResponseField name="data[].name" type="string">Nome (ex: "Mensal", "Anual").</ResponseField>
<ResponseField name="data[].payment_type" type="string">`one_time` ou `subscription`.</ResponseField>
<ResponseField name="data[].price" type="string">Preço decimal.</ResponseField>
<ResponseField name="data[].currency" type="string">ISO 4217 uppercase (`BRL`, `USD`).</ResponseField>
<ResponseField name="data[].billing_period" type="string|null">`month`, `quarter`, `semester`, `year` (apenas `subscription`).</ResponseField>
<ResponseField name="data[].schedule_iterations" type="integer|null">Quantas cobranças antes de encerrar (assinatura limitada).</ResponseField>
<ResponseField name="data[].is_active" type="boolean">Ativa.</ResponseField>
