---
title: "Criar oferta"
api: "POST /v1/offers"
description: "Cria uma oferta (preço/plano) vinculada a um produto."
---

**Escopo:** `offers:write`
**Idempotency:** suportada

## Body params

<ParamField body="product_id" type="integer" required>
  ID do produto (deve pertencer à tenant).
</ParamField>

<ParamField body="name" type="string" required>1-120 chars.</ParamField>

<ParamField body="payment_type" type="string" required>
  `one_time` ou `subscription`.
</ParamField>

<ParamField body="price" type="number" required>Preço decimal (≥ 0).</ParamField>

<ParamField body="currency" type="string" required>
  ISO 4217 (3 letras). Salvo em uppercase. Ex: `BRL`, `USD`.
</ParamField>

<ParamField body="billing_period" type="string">
  `month`, `quarter`, `semester`, `year`. Obrigatório quando `payment_type=subscription`.
</ParamField>

<ParamField body="schedule_iterations" type="integer">
  Número de cobranças antes da assinatura encerrar. Use para parcelamentos com fim definido.
</ParamField>

<ParamField body="is_active" type="boolean">Default `true`.</ParamField>

<RequestExample>
```bash cURL
curl https://seu-dominio/api/v1/offers \
  -H "Authorization: Bearer aut_..." \
  -H "Idempotency-Key: offer-mensal-php-001" \
  -H "Content-Type: application/json" \
  -d '{
    "product_id": 10,
    "name": "Mensal",
    "payment_type": "subscription",
    "price": 199.90,
    "currency": "BRL",
    "billing_period": "month"
  }'
```

```javascript Node.js
await fetch('https://seu-dominio/api/v1/offers', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer aut_...',

    'Content-Type': 'application/json',
  },

    name: 'Anual',
    payment_type: 'one_time',

```
</RequestExample>

## Limites

- 50 ofertas por produto.
- 500 ofertas por tenant.

<Note>
  Após criar a oferta, vincule-a a um **checkout** existente para começar a vender. A criação de checkout via API ainda não está disponível — use o painel.
</Note>
