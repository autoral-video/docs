---
title: "Listar aulas do módulo"
api: "GET /v1/modules/{module}/lessons"
description: "Lista paginada de aulas de um módulo."
---

**Escopo:** `modules:read`

## Path params

<ParamField path="module" type="integer" required>ID do módulo.</ParamField>

## Query params

<ParamField query="per_page" type="integer" default="25">1-100.</ParamField>

## Response

<ResponseField name="data[].id" type="integer">ID.</ResponseField>
<ResponseField name="data[].module_id" type="integer">FK do módulo.</ResponseField>
<ResponseField name="data[].type" type="string">`video`, `article` ou `quiz`.</ResponseField>
<ResponseField name="data[].title" type="string">Título.</ResponseField>
<ResponseField name="data[].slug" type="string">Slug.</ResponseField>
<ResponseField name="data[].description" type="string|null">Descrição.</ResponseField>
<ResponseField name="data[].duration_seconds" type="integer">Duração total.</ResponseField>
<ResponseField name="data[].sort_order" type="integer">Ordem.</ResponseField>
<ResponseField name="data[].is_active" type="boolean">Ativa.</ResponseField>
<ResponseField name="data[].is_free" type="boolean">Aula gratuita (preview).</ResponseField>
