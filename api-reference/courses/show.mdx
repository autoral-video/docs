---
title: "Detalhes do curso"
api: "GET /v1/courses/{course}"
description: "Retorna um curso específico."
---

**Escopo:** `courses:read`

## Path params

<ParamField path="course" type="integer" required>
  ID do curso.
</ParamField>

<RequestExample>
```bash cURL
curl https://seu-dominio/api/v1/courses/1 \
  -H "Authorization: Bearer aut_..."
```
</RequestExample>

<ResponseExample>
```json 200
{
  "data": {
    "id": 1,
    "name": "Curso de PHP",
    "slug": "curso-de-php",
    "description": "Do básico ao avançado.",
    "is_active": true,
    "modules_count": 8,
    "created_at": "2026-01-15T10:00:00+00:00",
    "updated_at": "2026-04-20T14:30:00+00:00"
  }
}
```

```json 404
{
  "error": {
    "type": "not_found",
    "title": "Recurso não encontrado.",
    "status": 404,
    "request_id": "..."
  }
}
```
</ResponseExample>

<Note>
  Cursos de outros tenants retornam **404** — não 403 — para não vazar existência.
</Note>
