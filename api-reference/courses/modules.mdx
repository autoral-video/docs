---
title: "Listar módulos do curso"
api: "GET /v1/courses/{course}/modules"
description: "Lista paginada de módulos de um curso."
---

**Escopo:** `courses:read`

## Path params

<ParamField path="course" type="integer" required>
  ID do curso.
</ParamField>

## Query params

<ParamField query="per_page" type="integer" default="25">1-100.</ParamField>
<ParamField query="page" type="integer" default="1">Length-aware.</ParamField>

## Response

<ResponseField name="data[].id" type="integer">ID.</ResponseField>
<ResponseField name="data[].course_id" type="integer">FK do curso.</ResponseField>
<ResponseField name="data[].name" type="string">Nome.</ResponseField>
<ResponseField name="data[].description" type="string|null">Descrição.</ResponseField>
<ResponseField name="data[].sort_order" type="integer">Ordem.</ResponseField>
<ResponseField name="data[].is_active" type="boolean">Ativo.</ResponseField>
<ResponseField name="data[].lessons_count" type="integer">Total de aulas.</ResponseField>
