---
title: "Listar checkouts"
api: "GET /v1/checkouts"
description: "Lista paginada (cursor) de páginas de checkout."
---

**Escopo:** `checkouts:read`

## Query params

<ParamField query="per_page" type="integer" default="25">1-100.</ParamField>
<ParamField query="active" type="boolean">Filtra ativos/inativos.</ParamField>

## Response

<ResponseField name="data[].id" type="integer">ID.</ResponseField>
<ResponseField name="data[].name" type="string">Nome.</ResponseField>
<ResponseField name="data[].slug" type="string">Slug (URL pública).</ResponseField>
<ResponseField name="data[].course_id" type="integer">Curso vinculado.</ResponseField>
<ResponseField name="data[].offer_id" type="integer">Oferta vinculada.</ResponseField>
<ResponseField name="data[].class_id" type="integer|null">Turma.</ResponseField>
<ResponseField name="data[].access_type" type="string">`lifetime`, `temporary`, `subscription`.</ResponseField>
<ResponseField name="data[].is_active" type="boolean">Ativo.</ResponseField>
<ResponseField name="data[].sales_count" type="integer">Vendas totais.</ResponseField>
<ResponseField name="data[].payment_methods" type="array">Métodos aceitos.</ResponseField>
<ResponseField name="data[].allow_refund" type="boolean">Permite reembolso.</ResponseField>
