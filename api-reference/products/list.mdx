---
title: "Listar produtos"
api: "GET /v1/products"
description: "Lista paginada (length-aware) de produtos da tenant."
---

**Escopo:** `products:read`

Produtos representam o "SKU" vinculado a um curso. Cada produto pode ter múltiplas ofertas (preços/planos).

## Query params

<ParamField query="per_page" type="integer" default="25">1-100.</ParamField>
<ParamField query="page" type="integer" default="1">Página.</ParamField>
<ParamField query="course_id" type="integer">Filtra por curso.</ParamField>
<ParamField query="active" type="boolean">Filtra ativos/inativos.</ParamField>

## Response

<ResponseField name="data[].id" type="integer">ID.</ResponseField>
<ResponseField name="data[].name" type="string">Nome.</ResponseField>
<ResponseField name="data[].description" type="string|null">Descrição.</ResponseField>
<ResponseField name="data[].course_id" type="integer">Curso vinculado.</ResponseField>
<ResponseField name="data[].is_active" type="boolean">Ativo.</ResponseField>
<ResponseField name="data[].offers_count" type="integer">Total de ofertas.</ResponseField>
