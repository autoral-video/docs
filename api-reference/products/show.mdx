---
title: "Detalhes do produto"
api: "GET /v1/products/{product}"
description: "Retorna um produto específico."
---

**Escopo:** `products:read`

<ParamField path="product" type="integer" required>ID do produto.</ParamField>

<RequestExample>
```bash cURL
curl https://seu-dominio/api/v1/products/10 \
  -H "Authorization: Bearer aut_..."
```
</RequestExample>

<ResponseExample>
```json 200
{
  "data": {
    "id": 10,
    "name": "Curso PHP — Acesso Vitalício",
    "description": "Acesso permanente ao curso completo.",
    "course_id": 1,
    "is_active": true,
    "offers_count": 3,
    "created_at": "2026-01-15T10:00:00+00:00",
    "updated_at": "2026-04-20T14:00:00+00:00"
  }
}
```
</ResponseExample>

<Note>
  Produtos de outros tenants retornam **404**.
</Note>
