description: "API REST pública da plataforma autoral.me — multi-tenant, escopo-restrita, com rate-limit por plano."
---

A API da **autoral.me** permite que tenants integrem programaticamente sistemas externos (CRMs, automações n8n/Zapier, ERPs, dashboards próprios) com a plataforma de cursos.

## Características

<CardGroup cols={2}>
  <Card title="REST + JSON" icon="code">
    Versionada em `/api/v1`, métodos HTTP padrão, respostas JSON.
  </Card>
  <Card title="Bearer tokens" icon="key">
    Autenticação via Personal Access Tokens (Laravel Sanctum) com prefixo `aut_`.
  </Card>
  <Card title="Escopos granulares" icon="shield-check">
    21 escopos do tipo `recurso:ação` (ex: `courses:read`, `students:write`).
  </Card>
  <Card title="Tenant-isolated" icon="building">
    Cada token é amarrado a um tenant na emissão. Recursos de outros tenants retornam 404.
  </Card>
  <Card title="Rate limit por plano" icon="gauge">
    Limite de requisições/minuto definido pelo plano do tenant.
  </Card>
  <Card title="Idempotency" icon="repeat">
    Header `Idempotency-Key` opcional em endpoints de escrita.
  </Card>
</CardGroup>

## Base URL

```
https://{seu-dominio}/api/v1
```

Substitua `{seu-dominio}` pelo domínio da sua tenant (ex: `escola.autoral.me` ou domínio próprio configurado).

<Note>
  Diferente do painel web (resolvido por domínio), a API resolve o tenant a partir do **token** — não do `Host`. Por segurança, um token emitido em uma tenant nunca acessa recursos de outra, mesmo se chamado em outro domínio.
</Note>

## Recursos disponíveis

| Recurso | Endpoints | Escopos |
|---|---|---|
| Identidade | `GET /me` | — |
| Cursos | list, show, modules | `courses:read` |
| Módulos | lessons | `modules:read` |
| Aulas | show | `lessons:read` |
| Alunos | list, show, grant/revoke access | `students:read`, `students:write` |
| Acesso | CRUD | `access:read`, `access:write` |
| Pagamentos | list, show, refund | `payments:read`, `payments:refund` |
| Assinaturas | list, show, cancel | `subscriptions:read`, `subscriptions:cancel` |
| Cupons | CRUD | `coupons:read`, `coupons:write` |
| Leads | list, show, create | `leads:read`, `leads:write` |
| Checkouts | list, show | `checkouts:read` |
| Produtos | CRUD | `products:read`, `products:write` |
| Ofertas | CRUD | `offers:read`, `offers:write` |
| Certificados | list, show | `certificates:read` |
| Perguntas | list, show | `questions:read` |
| Webhooks | CRUD | `webhooks:read`, `webhooks:write` |
